# Rekentool

Just a simple tool to fresh up your memory. I've used this project to learn Vue.

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
